# Zapojenie káblov - Modulárne zdroje

Tento repozitár slúži ako zdroj informácií o zapojení káblov k mojim modulárnym a semi-modulárnym zdrojom. Vznikol tak, že som potreboval doplniť káble k môjmu zdroju a prepinovať existujúce.

Schémy zapojenia sú generované programom [WireViz](https://github.com/wireviz/WireViz). 
Nákresy tvaru fyzických konektorov  sú kreslené  programom [DrawIO](https://www.drawio.com/) konkrétne [DrawIO Desktop](https://github.com/jgraph/drawio-desktop.)

- **Zdrojové súbory** – Všeobecne platné schémy zapojenia a datasheety ku konektorom
- **Zdroje podľa modelu** – Špecifické informácie pre jednotlivé modely zdrojov
